01. 함수 : 선언적 함수

자바스크립트에서 가장 일반적으로 사용되는 함수(function) 정의 방식 중 하나입니다.
이 방식은 function 키워드를 사용하여 함수를 정의하고, 함수의 이름을 지정한 후 중괄호({}) 내에 함수의 동작을 작성합니다. 

{
                    function func() {
                        console.log("01. 함수가 실행되었습니다.")     
                    }
                    func();            
                 
}

02. 함수 : 익명 함수

함수를 정의할 때 함수 이름을 생략하고 바로 함수를 정의하는 것을 익명 함수라고 합니다.
익명 함수는 주로 변수에 할당되어 사용되거나 다른 함수의 매개변수로 전달될 때 사용됩니다. 

{
                    const func = function () {
                        console.log("02. 함수가 실행되었습니다.")
                            }
                    func();
}
결과 확인하기
02. 함수가 실행되었습니다.

03. 함수 : 매개변수 함수

매개변수 함수는 함수가 호출될 때 입력으로 받는 값을 가지고 작업을 수행하는 함수를 의미합니다.
함수 정의 시 괄호 안에 선언되며, 함수가 호출될 때 이 매개변수에 값을 전달하여 함수가 그 값들을 이용하여 작업을 수행하게 됩니다. 

{
                    // 선언적 함수
                    function func(str) {
                        console.log(str);
                    }
                    func("03. 함수가 실행되었습니다.");
        
                    // 익명 함수
                    const func1 = function (str) {
                        console.log(str);
                    }
                    func1("03. 함수가 실행되었습니다.");
                   
}
결과 확인하기
03. 함수가 실행되었습니다.

04. 함수 : 리턴값 함수

리턴값(return value)을 가지는 함수는 함수가 실행된 후에 호출자(caller)에게 값을 반환하는 함수를 가리킵니다. 

{
                    //선언적 함수
                    function func() {
                        return "04. 함수가 실행되었습니다.";
                    }
                    console.log(func());
        
                    // 익명 함수
                    const func1 = function () {
                        return "04. 함수가 실행되었습니다.";
                    }
                    console.log(func1());                  
}
결과 확인하기
04. 함수가 실행되었습니다.

05. 함수 : 매개변수 + 리턴값 함수

매개변수와 리턴값을 가지는 함수는 함수가 호출될 때 입력값을 받아들이고, 그 값들을 이용하여 작업을 수행한 후 결과값을 반환하는 함수를 말합니다. 

{
                    //선언적 함수
            function func(str) {
                return str;
            }
            console.log(func("05. 함수가 실행되었습니다."));

            // 익명 함수
            const func1 = function (str) {
                return str;
            }
            console.log(func1("05. 함수가 실행되었습니다."));
}
결과 확인하기
05. 함수가 실행되었습니다.

화살표 함수 : 선언적 함수

화살표 함수는 함수를 정의하는 데 사용되며, 기본적으로 익명 함수로서 변수에 할당되거나, 다른 함수의 매개변수로 전달될 수 있습니다. () => { ... } 형태로 작성되며, ()는 매개변수를 받을 수 있고, =>는 함수의 정의를 나타냅니다. 중괄호({}) 내에 함수의 동작이 작성되어 있습니다. 

{
                    // 선언적 함수
                    function func() {
                        console.log("06. 함수가 실행되었습니다.");
                    }
                    func();
        
                    func1 = () => {   // 화살표 함수로 변경
                        console.log("06. 함수가 실행되었습니다.");
                    }
                    func1();
}
결과 확인하기
06. 함수가 실행되었습니다.

07. 화살표 함수 : 익명 함수

화살표 함수는 주로 익명 함수로 사용됩니다. 즉, 함수에 이름을 지정하지 않고 변수에 할당하여 사용하는 것이 일반적입니다. 이렇게 하면 함수를 간단하게 선언하고 다른 함수의 인수로 전달할 수 있습니다. 

{
                    // 익명 함수
                    const func = function () {
                        console.log("07. 함수가 실행되었습니다.")
                    };
                    func();
        
                    // 익명 함수 --> 화살표 함수
                    const func1 = () => {
                        console.log("07. 함수가 실행되었습니다.")
                    };
                    func1();
}
결과 확인하기
07. 함수가 실행되었습니다.

08. 화살표 함수 : 매개변수 함수

화살표 함수는 매개변수를 가지고 있는 함수를 간결하게 작성할 수 있는 방법입니다. 매개변수 함수는 함수가 호출될 때 입력값을 받아들이는 함수를 의미합니다. 

{
                    // 선언적 함수
            func = (str) => {
                console.log(str);
            }
            func("08. 함수가 실행되었습니다.");

            // 익명 함수
            const func1 = (str) => {
                console.log(str)
            }
}
결과 확인하기
08. 함수가 실행되었습니다.
