01. 함수 : 선언적 함수

함수를 정의할 때 그 함수가 무엇을 하는지를 명시적으로 선언하는 방식 

{
    function func() {
        console.log("01. 함수가 실행되었습니다.")
    }
    func(); 
}
결과 확인하기
01. 함수가 실행되었습니다.

02. 함수 : 익명 함수

익명 함수는 이름이 없는 함수를 의미하고 이러한 함수는 주로 콜백 함수나 함수 표현식으로 사용됩니다.
함수가 필요하지만 나중에 재사용할 필요가 없는 경우에 익명 함수를 사용할 수 있습니다. 

{
                    const func = function () {
                        console.log("02. 함수가 실행되었습니다.")
        
                    }
                    func();
}
결과 확인하기
02. 함수가 실행되었습니다.

03. 함수 : 매개변수 함수

매개변수 함수는 함수가 호출될 때 입력으로 받는 값을 가지고 작업을 수행하는 함수를 의미합니다.
함수 정의 시 괄호 안에 선언되며, 함수가 호출될 때 이 매개변수에 값을 전달하여 함수가 그 값들을 이용하여 작업을 수행하게 됩니다. 

{
                    // 선언적 함수
                    function func(str) {
                        console.log(str);
                    }
                    func("03. 함수가 실행되었습니다.");
        
                    // 익명 함수
                    const func1 = function (str) {
                        console.log(str);
                    }
                    func1("03. 함수가 실행되었습니다.");
                   
}
결과 확인하기
03. 함수가 실행되었습니다.

04. 함수 : 리턴값 함수

리턴값(return value)을 가지는 함수는 함수가 실행된 후에 호출자(caller)에게 값을 반환하는 함수를 가리킵니다. 

{
                    //선언적 함수
                    function func() {
                        return "04. 함수가 실행되었습니다.";
                    }
                    console.log(func());
        
                    // 익명 함수
                    const func1 = function () {
                        return "04. 함수가 실행되었습니다.";
                    }
                    console.log(func1());                  
}
결과 확인하기
04. 함수가 실행되었습니다.
