01. 변수 : 데이터 저장

자바스크립트(JavaScript)에서 변수는 값을 저장하고 참조하기 위한 기본적인 요소입니다.
변수는 값을 저장할 때 사용되며, 그 값을 나중에 필요할 때 다시 사용할 수 있습니다. 

{
    let x = 100;
    let y = 200;
    let z = "javascript";
    
    console.log(x);  // lat 변수 x값에 저장되어 있는 값을 출력
    console.log(y);  // lat 변수 y값에 저장되어 있는 값을 출력
    console.log(z);  // lat 변수 z값에 저장되어 있는 값을 출력
}
결과 확인하기
100
200
javascript

02. 변수 : 데이터 저장 + 데이터 변경

자바스크립트에서 변수는 필요할 때 값을 읽고 변경할 수 있습니다. 

{
    {
        let x = 100;
        let y = 200;
        let z = "javascript";

        x = 300;
        y = 400;
        z = "jquery";

        console.log(x);
        console.log(y);
        console.log(z);
    }
}
결과 확인하기
300
400
jquery

03. 변수 : 데이터 저장 + 데이터 변경 + 데이터 추가

변수를 사용하면 데이터를 저장하고 변경 및 추가 할 수 있습니다. 

{
{
    let x = 100;
    let y = 200;
    let z = "javascript";

    x += 300;
    y += 400;
    z += "jquery";
                
    console.log(x);  // 300 let변수 x값에 저장되어 있는 100을 더한 값 출력
    console.log(y);
    console.log(z);
}
    
}
결과 확인하기
400
600
javascriptjquery

05. 배열 : 데이터 저장(여러개) : 표현방법 1 : 배열 선언

배열은 여러 개의 값을 하나의 변수에 저장하는 자료 구조 입니다.
Array 객체를 사용하여 선언하고 동일한 종류의 데이터를 순서대로 저장하고 접근 할 수 있습니다. 

  {
    const arr = new Array();

    arr[0] = 100;
    arr[1] = 200;
    arr[2] = "javascript";

    console.log(arr[0]);
    console.log(arr[1]);
    console.log(arr[2]);
}
결과 확인하기
100
200
javascript

09. 배열 : 데이터 저장(여러개) : 표현방법 5 : push() 메서드를 이용한 배열 방법

배열 메서드를 사용하면 배열을 검색, 정렬, 필터링, 변형 및 조작할 수 있습니다.

11. 배열 : 데이터 저장(여러개) : 표현방법 7 : 배열구조분해할당

배열구조분해할당은 배열에서 필요한 요소만 추출하여 변수에 바로 할당할 수 있도록 합니다. 

{
                {
                    const arr = [100, 200, "javascript"];
                    const [a, b, c] = arr;
        
                  
                    console.log(a);
                    console.log(b);
                    console.log(c);
                }
                {
                    const arr = [100, 200, [300, 400]];
                    const [a, b, [x, y]] = arr;
        
                   
                    console.log(a);
                    console.log(b);
                    console.log(x);
                    console.log(y);
                }
}
결과 확인하기

400

12. 배열 : 데이터 저장(여러개) : 표현방법 8 : 배열펼침연산자

배열 펼침 연산자(spread operator)는 ... 기호를 사용하여 배열의 요소를 개별적인 값으로 분리하거나 병합하는 역할을 합니다. 

{
    {
        const arr1 = [100, 200, "javascript"];
        const arr2 = [300, 400, "jquery"];
        const arr3 = [...arr1, 500];

        console.log(arr1);      // 100,200,javascript
        console.log(...arr1);   // 100200javascript
        console.log(...arr1, ...arr2);  //100200javascript300400jquery
        console.log(...arr3)            //100200javascript500
    }
}
결과 확인하기
100,200,javascript
100200javascript
100200javascript300400jquery
100200javascript500
